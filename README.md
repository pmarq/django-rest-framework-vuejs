# django-vuejs-04-drf-vuejs

Django Rest Framework e VueJS desacoplados.

![django-vue04.png](img/django-vue04.png)


```
mkdir backend
cd backend
pip install Django==2.1.12 \
djangorestframework \
django-extensions \
dj-database-url \
python-decouple \
django-cors-headers
```

vue create frontend
cd frontend
npm install axios
vue add vuetify
npm run serve
